## Part B：示例/例题/习题（Worked Examples）

### EX-01：服务所有人的假设陷阱

| 字段 | 内容 |
|------|------|
| **示例 ID** | EX-01 |
| **来源位置** | 第一段（"我发现很多老师怕的不是续费谈崩"至"实际上有个商业认知的盲区"） |
| **示例类型** | 概念演示 |

#### 题目原文
"我发现很多老师怕的不是续费谈崩，反而是让家长失望。害怕让别人失望其实背后就藏着一个假设，就是只要我足够好，我就能服务所有人。这个假设听起来很有职业素养，但实际上有个商业认知的盲区。"

#### 求解目标
识别"只要我足够好，我就能服务所有人"这一假设存在的认知错误。

#### 解题步骤
1. 观察现象：老师害怕让家长失望
2. 追溯假设：该心理背后隐含"只要我足够好就能服务所有人"
3. 识别盲区：该假设听起来职业，但忽视了商业逻辑（并非所有人都适合被服务）

#### 正确答案
该假设存在商业认知盲区——"足够好"不等于"适合所有人"，服务能力与客户筛选是两个维度。

#### 结构化条件
| 条件编号 | 条件内容 | 条件类型 |
|----------|----------|----------|
| C1 | 老师害怕让家长失望而非续费谈崩 | 初始状态 |
| C2 | 害怕失望背后隐含假设：足够好=服务所有人 | 约束条件 |
| C3 | 该假设表面合理（职业素养） | 约束条件 |

#### 学科结构（用于生成变式题）
| 维度 | 描述 |
|------|------|
| **核心逻辑关系** | 表面合理的假设 → 实际隐含的商业盲区（服务质量 ≠ 客户通用性） |
| **关键操作类型** | 假设识别与解构 |
| **可变参数** | 行业类型（教培/咨询/医疗）、表面假设的具体内容 |
| **结构不变量** | "表面职业素养掩盖商业盲区"的认知结构 |

### EX-02：留住不合适客户的多层成本

| 字段 | 内容 |
|------|------|
| **示例 ID** | EX-02 |
| **来源位置** | "害怕让家长失望的陷阱"章节（"表面上看不非就是自己多受点委屈"至"实际上是在对付费留下的优质客户不公平"） |
| **示例类型** | 模型分析 |

#### 题目原文
"表面上看不非就是自己多受点委屈，多花点精力安抚，但实际的成本结构远不止如此。第一层成本其实就是口碑的稀释。不合适的家长往往也是最难被满足的家长。你服务的越努力，他越觉得理所当然，一旦有任何不符合预期的地方，他的负向反馈反而会越多。那么第二层也就是最隐形的成本，就是你对核心客户的服务质量会在下降，因为资源是有限的。"

#### 求解目标
分析留住不合适家长的真实成本结构，识别两层隐性成本。

#### 解题步骤
1. 剥离表面成本：自己受委屈、花精力安抚（可见）
2. 分析第一层隐性成本：口碑稀释——难满足的家长投入越多，理所当然感越强，负向反馈越多
3. 分析第二层隐性成本：核心客户服务质量下降——资源有限，不合适客户多占用一分，优质客户少得一分

#### 正确答案
两层真实成本：① 口碑稀释（努力越多→负向反馈越多）；② 优质客户服务质量下降（资源挤占）

#### 结构化条件
| 条件编号 | 条件内容 | 条件类型 |
|----------|----------|----------|
| C1 | 表面成本：受委屈+花精力安抚 | 初始状态 |
| C2 | 不合适家长是最难满足的家长 | 约束条件 |
| C3 | 服务越努力→理所当然感越强→负向反馈越多 | 过程变化 |
| C4 | 资源总量有限 | 约束条件 |
| C5 | 不合适客户多投入→核心客户少投入 | 过程变化 |

#### 学科结构（用于生成变式题）
| 维度 | 描述 |
|------|------|
| **核心逻辑关系** | 表面成本（可见）+ 第一层隐性成本（口碑）+ 第二层隐性成本（资源挤占），构成完整成本结构 |
| **关键操作类型** | 多层成本结构分析；资源零和博弈分析 |
| **可变参数** | 具体行业、客户类型、资源类型 |
| **结构不变量** | "难满足客户的边际服务效用递减"+"有限资源的机会成本"双重成本结构 |

#### 题型特征标签

### EX-03：客户结构决定商业走向

| 字段 | 内容 |
|------|------|
| **示例 ID** | EX-03 |
| **来源位置** | "害怕让家长失望的陷阱"章节结尾（"这其实不是情绪问题，而是资源分配的问题"至"这才能让你的商业可持续"） |
| **示例类型** | 概念演示 |

#### 题目原文
"这其实不是情绪问题，而是资源分配的问题。很多老师把留住家长当成服务能力的证明，但在商业逻辑里面，客户结构决定了你能走多远。真正的服务能力不是服务所有人的能力，是识别谁值得服务，并把资源集中在这里的能力。这才能让你的商业可持续。"

#### 求解目标
重新定义"服务能力"，建立客户结构与商业可持续性的逻辑连接。

#### 解题步骤
1. 澄清问题本质：不是情绪问题，是资源分配问题
2. 解构错误认知："留住家长=服务能力的证明"
3. 建立正确逻辑：客户结构→商业走向
4. 重新定义服务能力：识别谁值得服务+集中资源=真正的服务能力

#### 正确答案
真正的服务能力 = 识别目标客户 + 集中资源服务 → 客户结构优化 → 商业可持续

### EX-05：服务过程中边界显性化方法

| 字段 | 内容 |
|------|------|
| **示例 ID** | EX-05 |
| **来源位置** | "如何让不合适的家长'失望'"章节（"比如说在服务过程当中"至"边界从来没有被明确过"） |
| **示例类型** | 方法示范 |

#### 题目原文
"比如说在服务过程当中，大多数老师处理越界诉求时的本能反应是：怎么说才不得罪人？这个出发点本身就错了。问题不在措辞，在于你有没有一套清晰的服务标准。所以你真正要做的是把你的服务逻辑显性化。什么在范围内？什么不在？为什么？家长很多时候越界不是因为他们素质低，而是因为边界从来没有被明确过。"

#### 求解目标
掌握处理客户越界诉求的根本方法——建立显性化服务标准，而非优化措辞。

#### 解题步骤
1. 识别错误本能：优化措辞（"怎么说才不得罪人"）
2. 诊断根本问题：没有清晰的服务标准
3. 找到越界根源：边界从未被明确（非客户素质问题）
4. 正确操作：将服务逻辑显性化（三问：什么在范围内？什么不在？为什么？）

#### 正确答案
根本解：建立并公开清晰的服务标准（边界显性化），而非优化话术技巧

#### 结构化条件
| 条件编号 | 条件内容 | 条件类型 |
|----------|----------|----------|
| C1 | 常见错误本能：优化措辞 | 初始状态 |
| C2 | 问题本质：缺乏清晰服务标准 | 约束条件 |
| C3 | 越界原因：边界未被明确（非素质问题） | 初始状态 |
| C4 | 正确操作：显性化服务逻辑（范围内/外/原因） | 过程变化 |

| 维度 | 描述 |
|------|------|
| **核心逻辑关系** | 错误归因（话术问题）→ 正确归因（标准缺失）→ 根本解（边界显性化）；三层递进结构 |
| **关键操作类型** | 根本原因分析；标准显性化 |
| **可变参数** | 行业类型、越界诉求具体内容、服务标准内容 |

#### 题目原文
"现在你能明白，一个从不让任何家长失望的老师通常也没有真正的客户边界。没有边界就没有定位，没有定位就没有定价权，没有定价权就只能靠价格战竞争。所以让不合适的家长失望，不是你的服务出了问题，是你的筛选系统在正常运转。能做到这一点你才算是真正理解了客户关系这门手艺。"

#### 求解目标
分析"不让任何家长失望"行为模式导致的连锁商业风险，并重新解读"让家长失望"的正面意义。

#### 解题步骤
1. 起点：从不让家长失望 = 没有客户边界
2. 风险链：无边界 → 无定位 → 无定价权 → 价格战竞争
3. 重新解读：让不合适家长失望 = 筛选系统正常运转（非服务出问题）
4. 结论：真正理解客户关系 = 能做到让不合适家长失望

#### 正确答案
风险链：无边界→无定位→无定价权→价格战；让不合适客户失望是筛选系统健康运转的表现。

#### 结构化条件

| C2 | 无边界→无定位（因果） | 过程变化 |
| C3 | 无定位→无定价权（因果） | 过程变化 |
| C4 | 无定价权→价格战竞争（因果） | 终止状态 |
