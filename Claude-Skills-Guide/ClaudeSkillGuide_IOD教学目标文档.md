# 教学目标文档（Instructional Objectives Doc）

## 课程信息

| 字段 | 内容 |
|------|------|
| **课程名称** | The Complete Guide to Building Skills for Claude |
| **课程简介** | 本指南全面介绍了如何为 Claude 构建 Skill（技能），涵盖技能的基础概念、设计原则、使用场景规划、文件结构要求（YAML 与 Markdown）、测试与迭代方法、分发方式以及常见的五种高级模式与故障排除指南。 |

### 学习者先备能力要求

| 维度 | 内容 |
|------|------|
| **目标学习者** | 希望让 Claude 持续遵循特定工作流的开发者、高级用户以及寻求在组织内标准化 Claude 工作方式的团队。 |
| **学科前置知识** | 熟悉 Markdown 语法和 YAML 基础；对大语言模型的系统提示词有基本概念。 |
| **学科/工具能力** | 具备基本的文件编辑能力；如果涉及 MCP（Model Context Protocol）增强，需具备运行或构建 MCP server 的能力。 |
