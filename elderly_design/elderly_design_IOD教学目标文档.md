# 教学目标文档（Instructional Objectives Doc）

## 课程信息

| 字段 | 内容 |
|------|------|
| **课程名称** | 老年人支持性居家环境设计原则 |
| **课程简介** | 讲解如何通过色彩、材质和光环境的精确设计，为老年人构建一个安全、舒适的“支持性”居家环境，涵盖四个核心设计原则：消除地面视觉“陷阱”、加强色彩对比、规避引发幻觉的图案、打造柔和均匀的照明。 |

### 学习者先备能力要求

| 维度 | 内容 |
|------|------|
| **目标学习者** | 室内设计师、适老化改造从业者、有老人家居改造需求的家属 |
| **学科前置知识** | 基本的室内设计常识、对老年人生理特征（如视力退化）的初步认知 |
| **学科/工具能力** | 无特定要求 |

## Part B：示例/例题/习题（Worked Examples）

### EX-01：波打线地面防跌倒设计

| 字段 | 内容 |
|------|------|
| **示例 ID** | EX-01 |
| **来源位置** | 设计原则1 |
| **示例类型** | 概念演示 / 规则说明 |

#### 题目原文
避免使用在视觉上会产生高低差或台阶错觉的地面铺设（如波打线），以防老人因误判而摔倒。

#### 求解目标
识别并消除地面设计中可能导致老年人视觉误判的元素。

#### 解题步骤
1. 评估地面拼花或材质交界处是否有强烈的线条对比（如波打线）。
2. 分析该图案在老年人退化的视觉中是否会产生“台阶”或“坑洞”的三维错觉。
3. 若存在风险，改为平滑、无强烈边界切割的纯色或低对比度材质铺设。

#### 正确答案
取消波打线等易产生视觉高低差的地面铺设。

#### 结构化条件
| 条件编号 | 条件内容 | 条件类型 |
|----------|----------|----------|
| C1 | 存在地面铺设（如波打线） | 初始状态 |
| C2 | 老年人视觉对高低差易产生误判 | 约束条件 |

#### 学科结构（用于生成变式题）
| 维度 | 描述 |
|------|------|
| **核心逻辑关系** | 视觉图案特征 -> 三维空间错觉 -> 行走行为误判 -> 跌倒风险 |
| **关键操作类型** | 消除视觉陷阱边界 |
| **可变参数** | 波打线、拼花地砖、门槛石过门线 |
| **结构不变量** | 保持地面视觉连贯平坦，无欺骗性三维特征 |

#### 题型特征标签
#地面防跌倒 #视觉错觉规避 #适老化铺装

### EX-02：座椅色彩高对比搭配

| 字段 | 内容 |
|------|------|
| **示例 ID** | EX-02 |
| **来源位置** | 设计原则2 |
| **示例类型** | 方法示范 |

#### 题目原文
通过高对比度的颜色搭配（如给浅色椅子配上鲜艳的靠垫），帮助老人更清晰地辨识家具和物品的位置，避免物品与背景融为一体而难以发现。

#### 求解目标
利用色彩对比度提高家具物品的可辨识性。

#### 解题步骤
1. 确认背景颜色（如浅色椅子或浅色墙面）。
2. 选择与背景形成高明度/色相差的鲜艳色彩（如鲜艳的靠垫）。
3. 将高对比色应用在关键交互节点（座位、靠背），使物体边界清晰可见。

#### 正确答案
给浅色椅子配上鲜艳的靠垫，形成高对比识别。

#### 结构化条件
| 条件编号 | 条件内容 | 条件类型 |
|----------|----------|----------|
| C1 | 家具/背景颜色相近（如浅色椅子） | 初始状态 |
| C2 | 老年人对比度敏感度下降，难以区分相近色 | 约束条件 |

#### 学科结构（用于生成变式题）
| 维度 | 描述 |
|------|------|
| **核心逻辑关系** | 增加目标与背景的色彩对比度 -> 提升视觉凸显性 -> 降低寻找认知负荷 |
| **关键操作类型** | 强化色彩对比 |
| **可变参数** | 浅色椅子配鲜艳靠垫、深色门配浅色把手、浅色桌面配深色水杯 |
| **结构不变量** | 目标物与背景之间必须存在显著色差 |

#### 题型特征标签
#色彩对比 #视觉辨识 #适老家具配色

### EX-03：复杂图案引发认知干扰

| 字段 | 内容 |
|------|------|
| **示例 ID** | EX-03 |
| **来源位置** | 设计原则3 |
| **示例类型** | 概念演示 / 规则说明 |

#### 题目原文
不要使用容易产生视觉晃动感或立体错觉的复杂图案（如黑白几何地砖、特定纹理的壁纸或地垫），以免对老人造成认知干扰或眩晕。

#### 求解目标
识别并避免使用会引发老年人眩晕或认知混乱的装饰图案。

#### 解题步骤
1. 审查软硬装材料表面的图案设计（地砖、壁纸、地垫）。
2. 筛查是否存在高频重复的黑白几何图案、波普艺术图案或具强烈三维错觉的纹理。
3. 替换为素色、低对比度微纹理或自然柔和图案的材料。

#### 正确答案
避免使用黑白几何地砖、特定纹理壁纸等复杂图案。

#### 结构化条件
| 条件编号 | 条件内容 | 条件类型 |
|----------|----------|----------|
| C1 | 界面存在图案装饰（如黑白几何地砖） | 初始状态 |
| C2 | 老年人前庭功能和视觉处理能力下降，易受密集图案干扰 | 约束条件 |

#### 学科结构（用于生成变式题）
| 维度 | 描述 |
|------|------|
| **核心逻辑关系** | 复杂/密集/高对比几何图案 -> 视觉晃动/立体错觉 -> 认知干扰/眩晕风险 |
| **关键操作类型** | 过滤/规避干扰性图案 |
| **可变参数** | 黑白几何地砖、波浪纹地毯、密集条纹壁纸 |
| **结构不变量** | 界面图案需保持视觉稳定、不产生光流错觉 |

#### 题型特征标签
#眩晕规避 #认知干扰 #适老界面纹理

### EX-04：老人房均匀柔和主照明设计

| 字段 | 内容 |
|------|------|
| **示例 ID** | EX-04 |
| **来源位置** | 设计原则4 |
| **示例类型** | 模型分析 / 方案设计 |

#### 题目原文
不建议在老人房采用昏暗分散的“无主灯”设计，而应保证有明亮的主光源。同时，灯光应柔和且能缓慢亮起，以适应老年人较弱的视力和较慢的明暗适应能力。

#### 求解目标
为老人房配置适合其生理特点的照明系统。

#### 解题步骤
1. 评估照明架构：摒弃单纯追求氛围但照度不均的“无主灯”设计。
2. 配置主照明：在空间中心或主要活动区设置明亮的主光源，确保整体空间照度充足且均匀。
3. 优化光源质感：选择带柔光罩或防眩光的灯具，避免直射刺眼。
4. 增加缓亮控制：配置带有缓亮启动功能的驱动模块，让灯光开启时有一个渐亮过程。

#### 正确答案
采用明亮的主光源替代昏暗分散的无主灯，且保证灯光柔和、具备缓慢亮起的功能。

#### 结构化条件
| 条件编号 | 条件内容 | 条件类型 |
|----------|----------|----------|
| C1 | 老人房需要照明设计 | 初始状态 |
| C2 | 老年人视力较弱，需要更高照度 | 约束条件 |
| C3 | 老年人明暗适应能力变慢，忌突变强光 | 约束条件 |

#### 学科结构（用于生成变式题）
| 维度 | 描述 |
|------|------|
| **核心逻辑关系** | 生理机能退化（视力弱+明暗适应慢） -> 对应物理环境需求（高均匀照度+柔和防眩光+渐变点亮） |
| **关键操作类型** | 照明系统重构与灯控优化 |
| **可变参数** | 老人卧室主灯、起居室吸顶灯、卫生间照明 |
| **结构不变量** | 照度必须充足均匀，点亮过程必须平缓，光源必须防眩 |

#### 题型特征标签
#适老光环境 #均匀照明 #缓亮设计 #无主灯弊端
