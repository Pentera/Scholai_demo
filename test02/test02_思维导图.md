# test02 思维导图

## 中心主题：test02 - Python 编程第一课

### 分支 1：Python 基础输出
- **核心概念**：print 函数像一个扩音器，能在终端显示内容。
- **具体操作**：在 VS Code 中输入 `print("Hello World")` 并运行。
- **学习目标**：掌握向控制台输出字符串的基本方法。

### 分支 2：变量与赋值
- **核心概念**：变量是一个装数据的“盒子”，变量名是标签。
- **具体操作**：
  - 赋值：`x = 10`
  - 读取：`print(x)`
- **学习目标**：理解变量的作用，并掌握其基础语法。
