## 章节 1：Hello World，你的第一行代码

### 教学目标 (OBJ-01)
学生将能够使用 print 函数在终端输出指定的字符串。

### 核心概念：什么是 print？
在 Python 中，`print()` 就像是一个扩音器，它能把你想说的话在终端（也就是屏幕下方的小黑框里）展示出来。

### 动手实践：输出 Hello World (关联 EX-01)
让我们打开 VS Code 编辑器。在这个简单的环境中，输入以下代码：

```python
print("Hello World")
```

输入完毕后，点击顶部的运行按钮。
大家可以看到，下方的终端里打印出了 `Hello World`。这就是你和计算机的第一次对话！

## 章节 2：认识变量与赋值

### 教学目标 (OBJ-02)
学生将能够理解变量的作用，并掌握基本的变量赋值操作。

### 核心概念：装数据的盒子
接下来是**变量**。你可以把变量看作是一个“盒子”，这个盒子专门用来装我们要使用的数据。给盒子贴上标签（变量名），然后往里面放东西（赋值），就是编程中最常见的操作。

### 动手实践：给变量赋值 (关联 EX-02)
让我们来看看在代码里如何操作一个变量：

```python
x = 10
print(x)
```

1. `x = 10`：我们在内存里找了个盒子，贴上标签 `x`，然后把数字 `10` 放了进去。
2. `print(x)`：我们让计算机把盒子 `x` 里的东西拿出来，通过扩音器打印在屏幕上。

运行之后，屏幕会输出数字 `10`。
这就是变量的本质——它帮助我们在程序运行期间暂存和调用数据。
