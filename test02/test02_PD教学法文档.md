# test02 教学法文档

## 基本信息
- **关联来源文档**：test02_VFD源文档.md
- **关联 IOD**：test02_IOD教学目标文档.md
- **教学风格概述**：通过直观演示和生活隐喻引入编程概念，强调实践操作和即时反馈。

## Part A：教学流程

### 阶段 1：引入与演示
- **阶段功能**：引入 Python 第一课，展示基础的输出语法
- **关联 OBJ/EX**：OBJ-01, EX-01
- **教学行为**：首先在黑板上写下目标，然后立刻切换到实际的编辑器（VS Code）进行演示。通过直接执行代码让学生看到结果（终端输出 Hello World）。
- **回忆线索**："大家好，今天我们来学习 Python 编程的第一课"；黑板与 VS Code 界面的切换。

### 阶段 2：概念深化
- **阶段功能**：讲解变量的概念和赋值方法
- **关联 OBJ/EX**：OBJ-02, EX-02
- **教学行为**：采用生活化隐喻，将变量比喻为装数据的盒子。结合代码示例 `x = 10; print(x)` 验证该概念，帮助学生建立正确的心理模型。
- **回忆线索**："变量就像一个盒子，用来装数据"。

## Part B：教学模式与方法

### 目标：OBJ-01（Python基础输出）
- **主要教学方法**：直接演示法（Direct Instruction）
- **实例**：通过 VS Code 展示 print 函数的编写、运行和输出全过程，强化"所见即所得"的学习体验。

### 目标：OBJ-02（变量与赋值）
- **主要教学方法**：隐喻教学法（Metaphorical Instruction）
- **实例**：用"装数据的盒子"来比喻内存中的变量，将抽象概念具体化。

## Part C：讲师教学常用语

- "大家好，今天我们来学习..."（开场引入）
- "大家可以看到..."（引导观察实验结果）
- "你可以把它看作是..."（引入概念隐喻）
