# 教学法文档（PD）

## 1. 导入（Introduction）
展示第一人称视角的皮划艇视频片段，引导学习者将自己代入画面中，体验“沉浸式”的自然之美。

## 2. 呈现（Presentation）
通过定格画面的方式，引导学习者从远、中、近三个层次观察构图元素：远处的桥、中景的绿树、近处的船头；同时提示他们注意背景环境音中的规律性水声，体会视觉与听觉的结合。

## 3. 练习（Practice）
提供一个类似的静止图片或短视频，要求学习者使用类似的感官词汇（如“朦胧”、“节奏性”）写出一段至少50字的场景描述。

## 4. 总结（Conclusion）
总结第一人称视角叙事中的感官叠加技巧，并强调它如何增强受众的沉浸感。
