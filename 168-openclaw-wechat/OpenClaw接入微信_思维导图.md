# OpenClaw 接入微信——官方 ClawBot 插件安装与使用教程

## 微信版本升级与插件入口定位
- 在应用商店中将微信升级至最新版本
- 进入微信"我"→"设置"→"插件"界面
- 检查列表中是否出现"微信 ClawBot"入口
- 如果未出现入口，从手机后台彻底关闭微信进程
- 重新打开微信，再次进入插件界面检查（触发灰度推送检测）

## 使用命令行一键安装 ClawBot 插件
- 确保本地已安装 Node.js 运行环境
- 打开本地终端（如 macOS Terminal 或 Windows PowerShell）
- 输入官方安装命令：npx -y <官方安装包名>
- 回车执行命令，等待安装进度完成
- 安装完成后等待终端弹出二维码

## 扫码绑定微信账号与 OpenClaw
- 确认当前要用于绑定 ClawBot 的目标微信号
- 打开目标微信的扫一扫功能
- 扫描本地终端上弹出的安装完成二维码
- 在手机微信弹出的授权提示中点击"连接"
- 确认手机进入 ClawBot 对话窗口，完成绑定

## 在微信中调用 OpenClaw AI 进行对话与资讯查询
- 打开手机微信中的 ClawBot 聊天窗口
- 发送"你好"等简单问候语验证基础连通性
- 发送具体的任务型请求（如"查询24小时最新AI资讯"）
- 接收并查看 AI 返回的结构化任务结果

## 理解 ClawBot 的使用限制与账号管理规则
- 单点绑定限制：一个微信号只能绑定一个 OpenClaw 实例
- 账号解绑与更换：更换绑定账号需重新执行安装扫码流程
- 设备支持限制：目前仅支持手机端微信，不支持电脑端微信
- Agent 切换方法：在终端运行 openclaw config 命令绑定指定 Agent
- 隐私与安全隔离：他人无法添加你的 ClawBot，也无法将其拉入群聊

## 评估 AI-IM 集成趋势对个人工作流的战略意义
- 识别 AI 嵌入常用 IM 平台的核心优势（不改变使用习惯、降低使用门槛）
- 评估基于 IM 的 AI 助手对个人日常任务执行效率的提升价值
- 判断早期采纳此类集成工具所带来的先发优势
